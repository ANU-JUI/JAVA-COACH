# 🚀 Java Coach — AI Java Interview Preparation Chatbot

Java Coach is a topic-focused AI chatbot designed to help developers **learn Java concepts, debug code, and prepare for technical interviews** through a conversational interface.

Instead of being a generic chatbot, Java Coach is purpose-built for Java learning and interview workflows, offering a smooth and product-oriented user experience.

---

## 🌐 Live Demo

👉 https://java-coach.onrender.com/

---

## ✨ Key Features

### 🎯 Topic-Specific AI Responses

The chatbot is constrained to Java-related queries using prompt design, ensuring focused and relevant answers.

### 💬 Conversational Learning Experience

* Structured AI responses
* Code block formatting
* Typing indicators
* Sticky chat input
* Smooth message transitions

### 🧠 Mock Interview Mode

Users can simulate real Java interviews where the chatbot:

* Asks structured technical questions
* Waits for user responses
* Provides feedback
* Continues the interview flow

### 📂 Session Persistence

* Previous conversations are stored locally
* Users can switch between sessions
* Enables continuous learning without losing context

### 🔐 Lightweight Login System

A modal-based login reduces friction and maintains context without full page navigation.

### ⚡ Quick Action Suggestions

Users can start interactions instantly with chips like:

* Ask Java
* Debug Java
* Mock Interview

### 📱 Fully Responsive Design

The interface adapts across desktop, tablet, and mobile screens while preserving usability and navigation patterns.

---

## 🧩 Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Custom CSS with responsive layout patterns
* **AI Integration:** External LLM API (prompt-controlled for Java specificity)
* **State Management:** React Hooks + Local Storage
* **Deployment:** Render

---

## 🎨 Product Thinking & UX Decisions

This project focuses heavily on **product experience**, including:

* Reducing onboarding friction with modal login
* Maintaining conversational continuity with session history
* Designing a developer-friendly dark theme
* Providing contextual suggestion chips
* Ensuring sticky input for uninterrupted chat flow
* Structuring AI responses for better readability

The goal was to build a chatbot that feels like a **real SaaS learning tool**, not just a demo interface.
